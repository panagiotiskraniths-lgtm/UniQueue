<p>
Η <b>UniQueue</b> είναι μια ιστοσελίδα που στοχεύει στη μείωση της ταλαιπωρίας των
φοιτητών, παρέχοντας ενημέρωσεις για τον χρόνο αναμονής σε βασικές υπηρεσίες
του πανεπιστημίου.
</p>

<p>
Με την ιστοσελίδα, οι φοιτητές μπορούν να γνωρίζουν αν υπάρχει μεγάλη ουρά
στη λέσχη ή στη γραμματεία
</p>

<p>
Η ιστοσελίδα βασίζεται στην αξιοποίηση απλών δεδομένων ώστε να βελτιωθεί η
καθημερινότητα των φοιτητών.
</p>
